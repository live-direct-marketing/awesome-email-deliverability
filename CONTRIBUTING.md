# Contributing to Awesome Email Deliverability

Thank you for your interest in contributing!

## Guidelines

- Search previous entries before adding a new one to avoid duplicates.
- Make an individual pull request for each suggestion.
- Use the following format: `[Tool Name](link) — Description.`
- Keep descriptions concise and factual.
- New categories or improvements to the existing categorization are welcome.
- Check your spelling and grammar.

## Quality Standards

- The tool should be actively maintained.
- Free tier or free tools are preferred, but paid tools with significant value are accepted.
- No affiliate links.

## Pull Request Process

1. Fork the repository.
2. Add your tool to the appropriate category in `README.md`.
3. Submit a pull request with a brief description of why the tool should be included.
